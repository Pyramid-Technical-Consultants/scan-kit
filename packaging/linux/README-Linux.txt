Extract and run Scan Kit on Linux (x86-64).

1. Extract this archive anywhere you like.
2. Open a terminal in this folder and run:

       ./scan-kit

   Or use your file manager to double-click ``scan-kit``.

On first launch, Scan Kit installs a menu entry and icon automatically.
After that you can start it from your applications menu like any other app.

Requires Ubuntu 22.04 or newer, or an equivalent Linux desktop with X11 or Wayland.
