# ai-scratch

Throwaway scratch space for the AI assistant (and humans) to drop one-off
diagnostic / analysis scripts written while reasoning about a problem — e.g.
comparing model fits, probing data shapes, or sanity-checking a calculation.

- Everything in this folder is **git-ignored** except this `README.md`.
- Nothing here is part of the test suite, the package, or a release build.
- Treat the contents as disposable; delete freely.

Put files like `diag_model.py`, `check_residuals.py`, etc. here instead of the
repository root so the project root stays clean.
